This program represents a linkedlist data structure, the ui allows the user to create a linkedlist that stores movies, you can add and delete movies from any position in the linked list via the name of the movie, and print the movies forwards, backwards or by genre.

LIST OF FILES:

LinkedList.c
main.c
movieDefs.h

EXACT COMPILATION COMMANDS:

make

./JamesMcConnella4make

LAUNCHING AND OPERATING INSTRUCTIONS:

Most instructions are provided onscreen as you navigate through the UI, this UI doesnt handle all invalid inputs very well, although the linked list functions do, so operate with care.
